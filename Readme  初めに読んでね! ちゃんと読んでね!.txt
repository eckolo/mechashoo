■　ゲームの説明
○概要
機体と武装の組み合わせをカスタマイズして宇宙を駆ける傭兵となって戦場を転々とするアクション風味STG。
操作感がふわっとしてますが、慣れればそれほど難しくないはず。
プレイ時間は寄り道無しで40分程度。

○操作説明
・メニュー画面など
十字キー		：選択個所移動
Z、Space、Enter		：決定、選択、文字送り
X、Esc			：キャンセル、文字送り
・戦闘時
十字キー		：自機移動
Z			：手持ち武装１　通常動作状態⇔非動作状態の切り替え
X			：手持ち武装２　通常動作状態⇔非動作状態の切り替え
C			：機体に直接装備した武装　通常動作状態⇔非動作状態の切り替え
WASD、Shift+十字キー	：自機照準の移動（設定で操作方式を選択可能）
Shift			：低速移動
Space			：動作中の武装の一時停止＆手持ち武装の特殊動作発動
Esc			：ポーズメニュー

○要素説明
概ね通常のSTG。
特殊な要素は概ね下記の通り。

・装甲と障壁
攻撃を受けるとまず障壁が減り、残り障壁量を超えるダメージを受けるとその分装甲が減る。
障壁は時間経過で回復するが、装甲は1ステージ完了までそのまま。
敵機も同じで、まず障壁を抜かないとダメージは入らない。
被弾時、障壁で受け切った場合は青く、装甲までダメージが通った場合は赤く機体が発光する。
画面最上部の赤いゲージが装甲で青いゲージが障壁。

・燃料
武装の可動には一定量の燃料が必要。
燃料は時間経過で回復するが、消費されると回復量は落ちる。
残り燃料が足りない場合、武装は不発する。
画面最上部の黄色いゲージが燃料。

・武装の動作
武装によって、手に持った場合と機体に直接取り付けた場合で動作の異なるものがある。
手に持った場合は特殊動作が可能。
特殊動作の方向性は武装によって異なる。

・武装や機体の入手
各依頼のクリア時に何らかの条件を満たすことで武装や機体の入手が可能。
条件の種類や数は依頼によって異なる。
